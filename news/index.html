CD-i Related News

This page will sumarize news events that are related to CD-i.


December 30, 2000
Ahead will enhance upcoming 5.5 version of its Nero CD-recording tool with CD-i specific features
Ahead Software GmbH is about to release the next version of its popular CD-recording tool Nero in February of 2001. This new 5.5 version will include a number of general improvements (such as a non-destructive audio editor and an advanced cover editor), but it will also include advanced Video-CD creation features. Nero now includes a great quality MPEG-1 encoder to convert your AVI files into Video-CD format. Furthermore, there's an advanced menu editor, which creates menu screens with small thumbail images that enable you to select the video or still image that you want to view. And since Nero includes the 4.1.1 CD-i application for Video-CD, all of these advanced Video-CD 2.0 menu options are also accessible on a CD-i player. This will even allow you to create great picture discs with a nice menu interface for playback on your CD-i player (and Nero provides for an option to store the original JPEG's on CD too)!

Especially for CD-i owners, Ahead will include an option to customize the behaviour of the CD-i application, so that users can change the color of the cursor, the shape of the cursor, the track that is supposed to be used for centering the image, the location of the time and track indication on screen, whether the application should automatically start playback upon startup and whether the application should interprete the Video-CD 2.0 menu files, or just play the videos.


December 29, 2000
New version of CD-i Filesystem Driver for Windows coming out soon
In a few weeks, Vahur Sinijarv will release a new version of the CD-i filesystem driver for Windows!! This new version of the software that allows you to read the contents of a CD-i disc on a PC (beware: it's not an emulator!) offers a number of new features, including:
  • Now includes a neat control applet that can be accessed in Windows' Control Panel.
  • You are now able to view the disc's PVD or Primary Volume Descriptor in the control applet, to access important information about the loaded CD-i disc(s).
  • The driver can be turned "off" in the control applet to allow Windows to try accessing a disc itself. The driver is still active in the background, remove the files to de-install it.
  • The previous versions did not find all entries from a directory. All files are listed now.
  • File times and attributes are now properly set.
  • The driver now supports more than one CD drive. It allows for more than one CD-i disc to be recognized at a time.
  • Next to IDE drives, the driver now also supports SCSI CD-ROM, CD-R and DVD-ROM drives.
    The new version will be officially distributed by the ICDIA on this site. Keep looking out for it!!
