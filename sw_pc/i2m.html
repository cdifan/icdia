PC/Windows Downloads - Drivers and utilities for I2M CD-i boards

On this page, you will find drivers and support files for ISA and PCI versions of the I2M MediaAuthor and MediaPlayback CD-i boards for PCs.


  • I2M Media Playback 1.5.1 for Windows

    • MediaPlayBack 1.5.1 is the Windows frontend for the ISA version of the Philips/I2M CD-i board for PCs. Note that this software is only of use when used with the proper hardware.
    Download mediaplayback151.zip (874 KB)

  • OptImage CD-i Playback 2.2.1 for Windows

    • MediaPlayBack 2.2.1 is the Windows frontend for the PCI version of the Philips/I2M CD-i board for PCs. Note that this software is only of use when used with the proper hardware. Windows 3.1 / 1995 version.
    Download cdiplayback221.zip (613 KB)

  • I2M CD-i Playback 2.2b3 for Windows 95

    • MediaPlayBack 2.2b3 is the Windows 95 frontend for the PCI version of the Philips/I2M CD-i board for PCs. Note that this software is only of use when used with the proper hardware. Windows 95 / 1997 version.
    Download cdiplayback22b3.zip (1061 KB)

  • Configuring the PCI MediaAuthor CD-i Authoring board

    • Instructions on how to install the PCI version of the Philips/I2M CD-i board in a Windows 95 system. Includes notes on how to run MediaMogul with it. In Microsoft Powerpoint format.
    Download i2m_pci.zip (194 KB)
