22ER9060
Karaoke Mixer

The Karaoke Mixer can be used to mix CD-i player audio with audio from up to three microphones connected via 6.3mm input jacks. Intended to be used with karaoke (sing-along) discs, usable with any CD-i player with an audio output.

22ER9071
RF Modulator

The RF modulator allows certain CD-i players with no build-in RF (or antenna) connector to connect to an older TV-set. Only suitable for players with a 5 Volt output connector such as the CDI 450, CDI 470, CDI 490 and FW380i.

22ER9120
IC Card for CDI 180

Memory card of 8 KB that can be used as removable data carrier in addition to the standard non-volatile memory. Only suitable for the early CDI 180 player.

22ER9155
CDI 450 Connection Kit

This Connection Kit contains all cables and an additional AC/DC adapter to allow for the CDI 450 and the CDI 550 to be used at various locations.

22ER9208
I/O Port Splitter

The port splitter allows all CD-i players with just one I/O port (refer to the CD-i Players Comparison Chart) to connect two controllers for multi-player games.

22ER9209
Audio Stereo Splitter

Set of splitter cables that allows players which are connected to a TV-set using the CVBS and Audio connectors to be connected to a hifi system as well.

EM2200
RC5 IR Receiver

IR receiver that can be connected to CD-i players with an RC5-input port (refer to the CD-i Players Comparison Chart).

SBA2135
Audio L-R Cable

Audio cable for connecting CD-i players to a TV or hifi system using the Audio connectors.

SBC1139

Controller Extension Cable
Extension cable that can be used to enlarge the cord of all standard CD-i Pointing Devices.

SBC3032 / SBC3033 / SBC3037
Karaoke Microphone

Microphone with 6.3mm Jack that can be used with the 22ER9060 Karaoke Mixer.

SBV1127
Video (CVBS) Cable

Video cable for connecting CD-i players to a TV using composite video.
