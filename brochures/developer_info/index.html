Brochures - CD-i developer information

On this page, you will find scans of the contents of the CD-i Developer Information Package that Philips made available to new CD-i developers. All of documents are in Portable Document Format, you need Adobe Acrobat Reader to view or print them.

Please note that the addresses mentioned in these documents are not valid anymore.


  • CD-i Developer Information Package

    Frontpage of the information package binder containing the documents listed below.

    Note: The address listed on this frontpage is not valid anymore.

    Download devinfo_binder.pdf (1 page PDF)
  • CD-i Welcome

    Welcome leaflet for new CD-i developers describing the documents below.

    Note: None of the addresses listed in this document are valid anymore.

    Download devinfo_welcome.pdf (1 page PDF)
  • CD-i Frequently Asked Questions

    Introductory technical document describing the technical concepts and features of CD-i.

    Note: None of the addresses listed in this document are valid anymore. See the CD-i FAQ 2020 section of this website for a more up-to-date list of frequently asked questions.

    Download devinfo_faq.pdf (10 page PDF)
  • CD-i Technical Summary

    Basic technical specification of CD-i Base-Case players and the Digital Video extension.

    Note: The full technical specifications including the "Green Book" are available on this website in the sections CD-i Full Functional Specification (Green Book) and Microware OS-9 Documentation.

    Download devinfo_techsummary.pdf (2 page PDF)
  • CD-i Development Environments

    Description of suggested development and production environments on Mac, Sun or PC.

    Note: Documentation for several of the tools and environments is available on this website in the sections Microware OS-9 Documentation and Authoring Manuals.

    Download devinfo_environments.pdf (2 page PDF)
  • CD-i Publications

    Listing of the technical notes, books, magazines and newsletters relevant to CD-i.

    Note: Many issues of The Interactive Engineer and most of the listed books are available on this website in the sections The Interactive Engineer, CD-i Books or OS-9 Books.

    Download devinfo_publications.pdf (2 page PDF)
  • CD-i Directory

    CD-i related companies and institutions and a list of CD-i developers who want to be known publicly.

    Note: None of the addresses listed in this document are valid anymore.

    Download devinfo_directory.pdf (15 page PDF)
