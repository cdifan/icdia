Authoring Manuals

On this page, you will find scanned pages from manuals for CD-i development equipment and software.


  • Software Installation

    • From OS-9 System Disk for the CDI 605 player - January 1994
    This chapter will learn you how to install the OS-9 utilities on a CDI 605 system with or without a harddisk connected to it.
    Download 605_swinstall.zip (706 KB)

  • Accessing the OS-9 Utilites

    • From OS-9 System Disk for the CDI 605 player - January 1994
    This chapter will learn you how to access the OS-9 utilities on a CDI 605 after they have been properly installed.
    Download 605_accessos9.zip (463 KB)

  • Setting Up Internet

    • From OS-9 System Disk for the CDI 605 player - January 1994
    This chapter will learn you how to connect a CDI 605 player to a computer network using an ethernet connection and then communicate over it using the Internet Support Package. Please note: Internet in this context does not refer to the worldwide Internet network but to a local network based on TCP/IP.
    Download 605_internet.zip (1517 KB)

  • Using The SCSI Driver Suite

    • From OS-9 System Disk for the CDI 605 player - January 1994
    This chapter will learn you how to use the SCSI driver suite to access devices connected through a SCSI bus, such as harddisks or tape drives.
    Download 605_scsidriver.zip (927 KB)

  • Using The PC File Manager

    • From OS-9 System Disk for the CDI 605 player - January 1994
    This chapter will learn you how to access PC (MS-DOS) formatted diskettes and harddisks on an CDI 605 system, and how to use the OS-9 utilities with them.
    Download 605_pcdos.zip (2000 KB)


  • OS-9 IFF Documentation

    • Optimage - December 1992
    This document contains a wealth of information about the CD-I IFF file format, which is a standard format for storing CD-i assets (images and audio). The contents include:
    • OS-9/IFF Support Library C Functions
    • CD-I IFF Specification: Version 0.99 6 January 1989
    • A Quick Introduction to IFF
    • "EA IFF 85" Standard For Interchange Format Files: August 1, 1987
    • Audio Interchange File Format: AIFF

    Download IFF.docs.pdf (202 KB)
