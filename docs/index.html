CD-i Technical Documentation / System

On this page, you will find several technical documents related to CD-i hardware components and other system related subjects. Most of them are in Portable Document Format, you need Adobe Acrobat Reader to view or print them.


  • MONO II Player Status

    • Philips Media Electronic Publishing - February 1994
    This document is a technical summary of the status of the MonoII players. It begins with some general information and then gives specific problem reports. If workarounds are known they are presented.
    Download mono2status.zip (428 KB - 1x PDF in ZIP)

  • MCD212

    • Motorola, Inc. - 1995
    This document describes in full detail the MCD212 Video Decoder and System Controller (VDSC) that is used in CD-i players. It delivers extensive information about CD-i's video features, including technical aspects such as memory handling.
    Download mcd212rev0.pdf (873 KB - PDF)

  • MCD221 Technical Summary

    • Motorola, Inc. - 1995
    This document contains a technical summary of the MCD221 CD-Interface and Audio Processor (CIAP) that is used in CD-i players. It delivers some information about CD-i's audio features and the way the system communicates with the CD drive.
    Download mcd221tsrev0.pdf (286 KB - PDF)

  • MCD251 Technical Summary

    • Motorola, Inc. - 1995
    This document contains a technical summary of the MCD251 MPEG1 Full Motion Video (FMV) decoder that is used in the VMPEG Digital Video Cartridge (DVC) for CD-i players. It delivers some information about CD-i's MPEG video features and the way the system communicates with the MPEG decoder.
    Download mcd251ts.pdf (1.11 MB - PDF)

  • MCD270 Technical Summary

    • Motorola, Inc.
    This document contains a technical summary of the MCD270 MPEG Integrated Video and Audio decoder that is used in the IMPEG Digital Video Cartridges (DVC) for CD-i players. It delivers some information about CD-i's MPEG video and audio features and the way the system communicates with the MPEG decoders.
    Download mcd270ts.pdf (376 KB - PDF)

  • M68000

    • Motorola Inc. - 1992
    This archive contains Programmer's Reference Manual and Errata documents for the M68000 Microprocessor family. The M68000 family consists of 16/32 bit general-purpose microprocessor devices with varying external bus sizes and a common base instruction set. The Green Book specifies an M68000-compatible main processor for all CD-i players; an actual MC68HC000 is used in the Sony IVO portable CD-i players. Download m68000.zip (1.97 MB - PDF + TXT in ZIP)

  • SCC68070

    • Philips Semiconductors - November 1990 / April 1993
    This archive contains Technical Information documents for the SCC68070 Highly Integrated Microprocessor. The SCC68070 consists of an 68000-compatible CPU core, a two channel DMA controller, two serial channels, three timers, an I²C interface and a serial interface. It is used in all Philips players except the CDI 370 portable.
    Download scc68070.zip (8.80 MB - 2x PDF in ZIP)
    Note: There are some inaccuracies and inconsistencies in these documents.

  • MC68341/MC68341V

    • Motorola, Inc. - November 1995
    This archive contains Product Information and Manual documents for the MC68341 Integrated CD-i Engine. The MC68341 consists of an 68020 CPU, a two channel DMA controller, two serial channels, a timer and a serial interface. It is used in the Philips CDI 370 / Goldstar GPI 1200 portable players and on the I²M cards.
    Download mc68341.zip (1.75 MB - 4x PDF in ZIP)

  • TMP68301A/TMP68303/TMP68305

    • Toshiba - 1994
    This archive contains Technical Information chapters for the TMP683xx Microprocessor with Integrated Peripherals models. These processors consists of an 68HC000 CPU core and model-dependent numbers of DMA controllers, serial and parallel interfaces and timers. The TMP68305 is used in the Kyocera Pro-1000S portable player.
    Download tmp683xx.zip (15.3 MB - 3x PDF in ZIP)

  • DSP56001

    • Motorola, Inc. - May 1998

    Download dsp56001.zip (689 KB - 4x PDF in ZIP)
