CD-i Player and Emulator Files for Downloading

Free and shareware files to help in the production of CD-i and VCD titles, and selected pmpro software updates.

*Note: Files which do not have an extension may be treated as text by your browser, which may display them in your browser window as gibberish. If this happens, download the file as source.
In Netscape:
You can produce the dialog box for saving a page to disk by clicking on a link with the Shift key held down (option key on Macintosh), or clicking and holding the mouse button down. Then choose "Save link as", choose "Source" as file type, and give it a name.
In Internet Explorer:
Doesn't seem to get confused by the lack of an name extension. There is a "Download link to disk" option, invokable with the click-and-hold technique if necessary.
