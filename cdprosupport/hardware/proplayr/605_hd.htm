605 Hard Drive Compatibility Issues



605s and termination power

Q: (Customer) reports that DV playback is affected by when the hard drive is
powered up vis-a-vis the 605. If the hard drive is powered up before the 605:
DV problems. Powering up the hard drive last solves it. Has anyone else heard of this?

A:" It's the TERMPWR isolation problem. If an external drive provides +5v for
TERMPWR, it feeds back into the SCSI interface and fouls up the power-on
reset logic. Remember, for a while the expansion board SCSI connectors had
the TERMPWR tooth removed from the connector. I suppose that it was assumed
that something else would supply TERMPWR in this case.
I've noticed reset problems on the 602 where if the external SCSI drive is
on and the 602 is off (the 602's front panel power LED is glowing but you
might not see it if there's a lot of ambient light) if you power on the 602
sometimes the CD controller does not get reset and gives 244 errors or
hangs. Also the console startup messages report "CD software release 0.0"
when this happens. I've seen the floppy act flakey, too. (A customer's)
605/FMV will not work either if the hard drive is powered before the 605.
I thought that cutting the TERMPWR tooth off was a reasonable solution since
there was no isolation on the +5v. Now is looks like the tooth is back, but
I noticed no obvious change to the board to isolate TERMPWR. It's obviously
not since reset is a problem."

Summary of hard drive models and the 605:
Known to work:
Quantum 40, 80, 100, 210, 240mb
Seagate Wren series
       Hawk series (11200, 31200, 12400)
       Barracuda series (12550, 15150,etc)
Syquest (all)
Fujitsu (M2266, others)
Conner (some models)

Untried:
Maxtor, DEC

Known to be problematic:
HP 975448, 975449
