ImageMaker Articles

ImageMaker can take MPEG .rtf or .mmd (.mpg) files and produce a disc image suitable for use with MediaMogul Simulation testing. The disc image can be used directly (e.g. CDIPlayback) or recorded to a CD-R.

Things to remember:
  • Files which you want "Greened", i.e. multiplexed DV files, should be dragged to the Green file area. They should have either ".mmd" or ".mpg" as a filename extension.
  • Files which are already greened, that you want CD-i sectorized should also be put in the Green file area. They must have ".rtf" as a filename extension.
  • EP lists, scripts, miscellaneous files should be put in the Yellow files area.
