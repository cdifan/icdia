audio2mpeg

audio2mpeg is a Macintosh application that converts PCM or SoundDesigner II® files to MPEG level 2.  Conversion time is about 40 to 1 on a Quadra 950.

Current audio2mpeg owners: see the Mac page in the ftp section for information about a new audio2mpeg engine, as well as utilities which can be used with it. This engine is FAT (runs native on a PowerMac) and has optimized code. In general, it will run at least twice as fast as the current shipping version.
