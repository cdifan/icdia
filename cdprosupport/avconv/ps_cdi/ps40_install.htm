Installing the PC CD-i Plugin
with Adobe Photoshop 4.0

Adobe changed the organization and nomenclature of their plugin folders with version 4.0. Thus the installer for the CD-i Plugin for Adobe Photoshop for Windows will not install the files in the correct locations. So a manual install is required instead:
  1. Create a temporary folder in which to install the CD-i Plugin files.
  2. Run the installer on the CD-i Plugin diskette. When prompted for the install path, type in the path to the temporary directory you just made.
  3. After the install is done, manually move the CD-i Plugin files to the correct Photoshop subfolders:
    1. Place the files ending in a in the /Plugins/Import subdirectory.
    2. Place the files ending in e in the /Plugins/Export subdirectory.
    3. Place the files ending in f in the /Plugins/Filters subdirectory.
  4. Delete the temporary folder.
