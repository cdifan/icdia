File "type" problems in Audio and Animation Stack

Both Audio and Animation Stack look at the "type" attribute of a file to determine if it should be displayed in the Open... dialog. If assets are imported from other platforms, they may not have the "type" set properly for Audio or Animation Stack to perform conversions. The solution to this dilemma is to use a utility to change the type to one of these possibilities:

Audio Stack:

  • AIFF
    For aiff files
  • Sd2f
    For SoundDesigner II files

     

Animation Stack:

  • PICS
    For PICS files
  • MooV
    For Quicktime Movie files

     

There are many utilities which can set a file's type on a Macintosh. Apple's ResEdit can, and do many other (more dangerous) things as well. A few other utilities in the Shareware area are SetItsType (an application), and my personal favorite GetMoreInfo. Do not change the type of a file unless you are sure that it is of that type. If the structure of the file is not valid, the stack and/or your Mac may crash.
