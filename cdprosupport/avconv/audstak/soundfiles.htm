Audio File Problems with Audio Stack

 

  Audio Stack sometimes rejects files, often with a curious directory error, if it dislikes some part of an audio file. This is especially common with SoundEdit AIFF files. In order to get around this, a few strategies are:

  1. Using SoundEdit, resave the file as an SDII instead of AIFF, and try again.
  2. Chop (or pad) a few milliseconds off front or back, resave and try again.
  3. Using different sound editing software, open the audio file and resave and try again. ZMac's Soundsmith software is a useful alternative, available on MacUser's website.
