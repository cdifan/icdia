Sun/UNIX Conversion Articles

There are two products available for asset conversion on the Sun:

ICU Image Conversion Utilities
These command-line utilities can convert convert the following file types:

Computer types:

  • Targa
  • Vista
  • RGB888

    CD-i types:

  • DYUV
  • Clut 4, 7
  • RunLength 3,7            to CD-i compatible formats.

    Special conversions, like animations and dyuv partials, are also possible.

    ACU Audio Conversion Utilities
    These command-line utilities can convert convert the following file types:

  • Raw PCM or
  • AIFF files          to CD-i ADPCM formats.

    The files must be at one of these sampling rates:

  • 44.1KHz
  • 37.8KHz
  • 18.9KHz

    The ACU's also contain a number of utilities for less commonly used conversions.
