MediaStockroom known bugs
  • Reported errors in sample rate conversion:A problem with the READINFO dll in MediaStockroom causes an incorrect error message during audio conversions. The file is actually converted correctly, but the error reports otherwise.

  • QHY Conversion bug: A problem with the VIDPAK dll can cause video anomalies with QHY conversions.

    • An update to both of these files to fix the problems is contained in a .zip file on this site: ms_dlls.zip. After unzipping, these files should be placed at the root of the MediaStockroom directory.
