Authoring Products - MediaMogul


MediaMogul CD/Docs
MediaMogul authoring software & Documentation. Requires PC or Mac CD-i Authoring Card or 605 player.

MediaMogul Documentation (Replacement-only)
MediaMogul documentation for replacing lost copies

MediaMogul Plug-in - Bar Code Reader
Enables MediaMogul to scan bar codes, like UPC bars, and store that data in a String Variable. The data can then be manipulated, displayed or used in branching via the String Variables plug-in. Requires Bar Code Reader hardware and compatible RS-232 interface. Requires String Variables plug-in.

MediaMogul Plug-in - Course Map/Graphics Blitter
Enables MediaMogul to track which sections of the title have been viewed/completed. Based on this data, MediaMogul can dynamically change the appearance of Menu pictures (like place a check mark next to completed selections) and control branching. Requires String Variables plug-in. Click here for more info...

MediaMogul Plug-in - Digital Video
Enables play of Digital Video (MPEG-1 moving video) in MediaMogul. This plug-in also permits still picture overlays and is cooperative with other plug-ins (like Pop-Up) to display on screen controls and overlaid menus while Digital Video is playing.Click here for more info...

MediaMogul Plug-in - Dynamic Hotspots
Enables MediaMogul to dynamically change the "click" action of a given hotspot. Often used where the "context" of a menu hotspot changes, like in menus. Requires String Variables plug-in.

MediaMogul Plug-in - File Access
Enables MediaMogul to read, write and update files on a floppy drive or hard drive attached to a CD-i player, or to files in Non-volatile player memory. Requires String Variables plug-in.Click here for more info...

MediaMogul Plug-in - Horizontal Scrolling Image
Allows MediaMogul to display an image much wider than the normal screen by scrolling it left/right. The scrolling image can have hotspots on it, as well as having hotspots outside the scrolling image.

MediaMogul Plug-in - Irregular Menu (ON MOGUL CD)
Permits menus that have hotspots which are not rectangular. With this plug-in, hotspots can be any shape, including geometric and highly irregular shapes (like continents in an Atlas title). DESCRIPTION ONLY - CANNOT BE PURCHASED SEPARATELY

MediaMogul Plug-in - Keyboard
Enables MediaMogul titles to receive input from a REAL keyboard connected to a CD-i player. The user can enter information or make selections without using the pointing device. The user navigation method is similar to that of a Video CD player. CD-i or ASCII RS-232 keyboard required.Click here for more info...

MediaMogul Plug-in - Kiosk Brancher & Virtual Keyboard
Contains : Menu Branch Manager to track and lock out areas of the title previously visited. Virtual Keyboard uses hotspots to represent keys of a keyboard (alpha and numeric) to permit the user to enter information. Eliminates the need for a physical keyboard. Requires String Variables plug-in.

MediaMogul Plug-in - Magnetic Stripe Reader
Enables MediaMogul to read Magnetic Strips, like those on credit cards, and store that data in a String Variable. The data can then be manipulated, displayed or used in branching via the String Variables plug-in. Requires Mag Stripe Reader hardware and compatible RS-232 interface. Requires String Variables plug-in.

MediaMogul Plug-in - Pointerless Menu (ON MOGUL CD)
Pointerless Menu lets you create a menu where the cursor-arrow "jumps" directly from one hotspot to another (no dead zone between hotspots). Often used where the remote control is used to navigate screen (not a precise as a mouse or trackball) . DESCRIPTION ONLY - CANNOT BE PURCHASED SEPARATELY

MediaMogul Plug-in - Pop-up Menu
Add "global" menus, like Digital Video play controls, which can be displayed (pop up from the bottom of the screen) at any time during a presentation/title, including during Digital Video. Pop-up menu can be used independently of, or in conjunction with, standard MediaMogul menus.

MediaMogul Plug-in - Printer Manager
Enables MediaMogul to print text and raw files (like Postscript), or dynamic items (like accumulated names and addresses) to a serial or parallel printer. Requires String Variables plug-in.

MediaMogul Plug-in - QHY Highlighting & Zooming
Displays a border around hotspots when the arrow-cursor enters the active area. QHY Zoom permits a rectangular area anywhere on the original QHY picture to be magnified.

MediaMogul Plug-in - Run-length Linked/Animations/Scroll
Contains: Run-Length Linked, where moving the arrow-cursor over a hotspot causes a run-length picture to be displayed immediately; Run-Length Animations, where moving the arrow-cursor over a hotspot causes an animation to play immediately; Run-Length Scroll, where moving the arrow-cursor over a hotspot causes a longer or wider than full screen picture to scroll immediately.

MediaMogul Plug-in - String Variables
Expands on MediaMogul's integer-only variable set. Permits MediaMogul to store, display and manipulate string data (like names, messages, numbers...). Also extends the maximum number of variables beyond MediaMogul's A-Z. Many other plug-ins require String Variables.Click here for more info...

MediaMogul Plug-in - Super Image Scroller
Enables MediaMogul to display pictures longer or wider than the screen, by scrolling them horizontally or vertically. The scrolling picture can have hotspots on it (in addition to any other hotspots concurrently on the screen). The plug-in can also "bookmark" the last selection made on the scrolling picture. Click here for more info...

MediaMogul Plug-in - Talking Menu Plug-in (ON MOGUL CD)
Plays small sound files when the cursor moves over a hotspot. Often used in childrens and language training titles. DESCRIPTION ONLY - CANNOT BE PURCHASED SEPARATELY

MediaMogul Plug-in - Test Engine
Enables MediaMogul to ask T/F, or multiple choice questions from easily assembled and changeable text files, record and evaluate responses, and control branching based on responses. Requires String Variables plug-in. Click here for more info...
