MediaMogul Studio Pak for Windows



pmpro sells a full-featured CD-i development package for Windows 3.x or Windows 95 machines. The CD-i development package has everything you need to make CD-i training, educational, presentation, and testing applications. Included are tools to: display still pictures, play audio, menus, digital video (MPEG), collect, manipulate, and display text and numbers. And, to accommodate features unique to your presentations, your choice of five special-feature add-ons (plug-ins).

MediaMogul Features:
  • GUI based tool which runs on the CD-i Authoring Board, and shares PC host hard drives, CD-drive and memory
  • Is a script-based language with easy to learn English keywords
  • Building software is native a Windows application for fast processing
  • Versatile playback of still and 30 fps, full-screen MPEG
  • Can collect, manipulate, display and print text, numbers and other dynamic data
  • Variety of video transitions, including curtain, wipe, dissolve and cut
  • Supports three levels of audio, including near-CD quality
  • Over 20 special-feature plug-ins available to expand functionality of your titles
The MediaMogul Studio Package for the PC includes:
  • Authoring Board for the PC (PCI or ISA)
  • CD-i Trackball for authoring board
  • MediaMogul v2.2.1 (CD-i authoring software)
  • Talking Menu Plug-in for MediaMogul
  • Irregular Menu Plug-in for MediaMogul
  • Screen Dimmer Plug-in for MediaMogul
  • String Variable Plug-in for MediaMogul
  • Constrain Cursor Plug-in for MediaMogul
  • Digital Video Plug-in for MediaMogul
  • Five Plug-ins of your choice
  • Entry Point Assist for Windows (creation of entrypoint files for use with Digital Video/MPEG)
  • CD-i Plug-in for Adobe Photoshop (CD-i graphics conversion)
  • MediaStockroom (CD-i graphic and audio conversion utility)
  • script2disc for Windows (assembles the CD-i disc image directly from the MediaMogul scripts)
  • CD recording software (records the disc from a CD-i disc image--hosted on the PC and used in conjunction with the CD-recorder)
  • CD recorder (external SCSI)
  • Ten blank CD-R's
  • $500 per person training credit
  • One year of fax/phone/email development support for entire package
