OS-9 "C" 605 Programmer Pak
CDI605 player with digital video cartridge and emulator for testing applications written in C. (Hard drive, keyboard, and monitor not included.) Optional products to use with this platform: disc building software, CD recording option, and C compiler package.

OS-9 CD Verifier Package
The Optical Disc Verifier Package consists of CD-i Disc Verifier, Video CD Disc Verifier, and Photo CD Disc Verifier. Each of these verifiers performs checks to assure that the disc layout and structure are in compliance with the CD-i, Video VC, and Photo CD specifications. A report is prepared by the tool containing all detected deviations from the standards.

OS-9 E2 Emulator
Designed for CD-i development on OS-9, the emulator provides an environment for testing a CD-i application without having to record a write once disc. The application can be accurately emulated from the hard disc. Requires CDI605 and hard disc. (Disc building and CD recording options not included.)

OS-9 I2M CD-i Authoring Card - PCI
CD-i card that inserts into a computer PCI slot (Mac or PC). Required to run MediaMogul and other authoring products. Card also will play CD-i discs, Video CD discs, disc images as well as preview CD-i pictures, CD-i audio or multiplexed and real-time format MPEG files. Card requires RGB or composite TV, multimedia speakers and CD-i mouse/trackball.

OS-9 I2M CD-i Playback Card - PCI
CD-i card that inserts into a computer PCI slot (Mac or PC). Enables computer to play CD-i discs, Video CD discs, disc images.
