Reinstalling CDIPlayback for Macintosh


If you have deleted parts of CD-i Playback for Macintosh, or are having trouble with your current installed copy, you may need to reinstall the software. In order to avoid problems, we have found it best to reinstall using this procedure:
  1. Delete all remnants of the old installation first. These include:
    • The CDIPlayback  application, which should be in your MediaMogul folder.
    • The !CDIPlayback Init, which is in the System:Extensions folder.
    • The CDIPlayback Stuff  folder, which also is in the System:Extensions folder.
    • The CDIPlayback Prefs  file which is in the System:Preferences folder.
  2. Reboot your Macintosh with the extensions off. This is accomplished by pushing the Shift key down until the message "Extensions Disabled" appears in the Welcome to Macintosh dialog.
  3. Reinstall the CD-i Playback software according to the manual.
  4. Reboot.
