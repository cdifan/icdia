CD-i Fonts

There are two kinds of fonts usable with CD-i: UCM and PRL bitmapped.

There are a few UCM fonts which are contained on the MediaMogul software cd, but these fonts are not as commonly used.

There a number of PRL fonts available from our Sales department as well. These are:

  • Blitt
  • Elsted
  • Garson
  • Grasset
  • Mickleham
  • Minster
  • Tring
  • CD-i Single Bit Fonts
  • Also, computer fonts can be converted to PRL fonts with a utility called FontCatcher.
