PINK MPEG-1 Stream Multiplexer

    PINK is an application program for SUN, Mac and PC.  It allows you to multiplex MPEG-1 video files with MPEG audio files.  The output is a single file containing the interleaved video and audio data with an MPEG system layer.  The process is controlled by a PINK script.
