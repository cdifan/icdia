Entrypoint Assist


Entrypoint Assist is a utility program designed to create ancillary files for use with the Digital Video Plug-In. Entrypoint Assist reads multiplexed MPEG files or realtime files containing MPEG streams and produces:

  • Entrypoint files in several formats
    • Digital Video Plug-In for MediaMogul 2.2
    • Digital Video Plug-In for MediaMogul 2.1
    • Pink entrypoint list
    • Philips VCD entrypoint list
  • Delta files *
  • Log files

* Please note that Delta files are only needed for use with the OS-9 versions of the master software.

There are versions of Entrypoint Assist available for Macintosh, PC (Windows), and CD-i player platforms.

If no selections are made in the Entrypoint Assist window (Macintosh or PC versions) or no options are entered in the command line (CD-i player version) all the above files will be produced when Entrypoint Assist is run. Each of the files produced will have the same base filename as the source file with the following extensions:

Digital Video Plug-In for MediaMogul 2.2    source.mm
Digital Video Plug-In for MediaMogul 2.1 source.ir
Pink entrypoint list source.pkl
Philips VCD entrypoint list source.pil
Delta file source.dlt
Log file source.log

The default behavior of Entrypoint Assist is to produce a file containing all entrypoints from the source MPEG. If you require fewer entrypoints, this can be modified by entering a default number of seconds between entrypoints. The minimum number of seconds between entrypoints is entered in the Macintosh and Windows versions. In the CD-i player version, this is entered as an option on the command line using the -s seconds option, where seconds is the number of seconds desired between entrypoints.

The log file produced by Entrypoint Assist is useful for determining information about the entrypoints are included in the entrypoint file. It contains a listing of the number of the entrypoint, the SMPTE timecode, the raw sector number, byte offset, and pack offset for each included entrypoint.

For further information about Entrypoint Assist, please refer to the manual which you received along with your Digital Video Plug-In.
