Delta Vc Model 20 (PC based) MPEG-1/2 Encoder

    DeltaV20 is a high end MPEG-1 and MPEG-2 encoding system. It runs on a Pentium PC under Windows 3.x. It includes software which allows you to make Video CD 2.0 compliant disc images. These disc images can then be recorded to CD recordable media.
