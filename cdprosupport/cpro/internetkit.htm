Internet Toolkit for CD-i

Internet Toolkit is a C Library, containing API's to do everything from managing dial-out, connections via SLIP/PPP, TCP/IP communications and FTP file transfers, Web browsing, News Group management, and email. Modules include:

Basic Toolkit:
* managing dial-out (Tele-CD-i code)
* can store user account (ISP info) in NVRAM
* supports most Hayes 14.4k or 28.8k modems at up to 19k baud (serial port rate)
* ISP connections via SLIP/PPP
* TCP/IP protocol
* manual or automatic ISP logon (IP address, dynamic name servers, name, password...)
* FTP file transfers

Web Browsing Option:
* HTTP, HTML 2.0 compliant browser
* supports some HTML 3.0 features--FORM, INPUT TYPE, TEXTAREA, PASSWORD, CHECKBOX, RADIO, IMAGE, SUBMIT, REST, SELECT, OPTION
* decodes GIF and JPEG images into CLUT or DYUV as appropriate.
* hotspots and URL tagging
* conditional link branching
* scrolling browser window
* bookmarking
* RAM cacheing

Email Option:
* send or receive messages
* can use virtual or physical keyboard
* can format text as plain text or in a Web Browser window
* POP3 (server-based email) and SMTP (simple mail) protocols

News Groups Option:
* display list of news groups
* send or retrieve headers and messages
* NNTP (News) compliant

Modem Requirements:
* External RS-232
* 14.4k or 28.8k baud
* any of; Bell 103/212A, V.21, V.22, V.22bis, V.23, V.32, V.32bis
* error correction/compression, any of; V.42bis, MNP 2-5, MNP 10
* can have Group III Class II fax features (not necessarily supported in the toolkit)

Authoring Requirements:
* Microware K&R compiler (NOT Ultra C)
* PC or Sun workstation
* knowledge of 0S-9 system and Green Book

Playback Station Requirements:
* CDI-Player with one serial port
* DV cartridge (needed for the extra 1 Meg RAM)
