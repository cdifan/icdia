OS-9 Disk Images Downloads

On this page, you will find the contents of several floppy disks in Disk Image format. These files can be written to a physical 3.5" DDSD (Double Density Double Sided) floppy for use in CD-i players with floppy drivers or indeed any other OS-9 system.

The Disk Image files are in OS-9 RBF format and their contents can be viewed, extracted and otherwise manipulated with the "os9" program that is included in ToolShed.

SCP image files made with a GreaseWeazle FLUX reader are available on archive.org and should be writable to a 3.5" DSDD floppy using a FLUX-level floppy writer (eg. SuperCard Pro, KryoFlux, etc).

If needed, SCP image files can be converted to different formats with the HxCFloppyEmulator program.


  • OS-9 System Diskette (for CDI 605 player)

    • February 10, 1993
    With this disk you can obtain an OS-9 command prompt on a serial terminal connected to a CD-i 605 player by selecting LOAD in the SETTINGS screen. The disk provides essential OS-9 utilities to customers who own a CD-i 605 player. The contents of this disk are described in OS-9 System Disk for the CDI 605 player in the Authoring Manuals section.
    Download os9sys.dsk (638 KB), SCP image available on archive.org.

  • OS-9 System Diskette (adapted for CDI 18x player)

    • February 10, 1993
    With this disk you can obtain an OS-9 command prompt on a serial terminal connected to the 182 unit of a CD-i 18x player by selecting CD-RTOS in the SYSTEM screen. The disk was adapted from the OS-9 System Diskette above by changing the startup script.
    Download os9sys180.dsk (638 KB), SCP image available on archive.org.

  • 605 Driver Update

    • February 10, 1993
    This disk contains updated driver modules, the PC-DOS file manager (PCF) and the latest version of the Internet Support Package for the CD-i 605 player. The updates resolve a number of known problems with the current drivers in ROM.
    Download 605upd.dsk (638 KB), SCP image available on archive.org.

  • OptImage Public Domain Software 1 of 2

    • February 10, 1993
    This disk contains the following utilities for usage on your CD-i player or OS-9 development system with a floppy drive:
    bfed da kermit du iffinfo loadim.ev umacx namemod toos9 vi volname.
    Download pubsoft1.dsk (638 KB), SCP image available on archive.org.

  • OptImage Public Domain Software 2 of 2

    • February 10, 1993
    This disk contains C sources for the CD-i IFF library and a PostScript version of the OS-9 IFF documentation also available in the Authoring Manuals section.
    Download pubsoft2.dsk (638 KB), SCP image available on archive.org.
