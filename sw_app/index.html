CD-i Applications

On this page, you will find several CD-i runtime applications for Video-CD and Photo-CD. These applications can be included on the CD so that the CD-i player will use them when a Video-CD or Photo-CD is played on a CD-i player. Refer to CD-i Technical Documentation / Software for documentation about these applications.


Video-CD

  • Video-CD on CD-i 3.2

    • Version 3.2 is the latest version of the Video-CD 1.1 runtime for CD-i. It is prefered to use this application instead of version 4.x when no Video-CD 2.0 functionality is required. Refer to question 6.3.1 in the FAQ for more information about the various versions of this application.
    Download vcd_on_cdi_32.zip (720 KB)

  • Video-CD on CD-i 4.1.1

    • Version 4.1.1 is especially developed to incorporate all of the enhanced Video-CD 2.0 functionality. It supports the Video-CD 2.0 methods for displaying menus with number selection or hotspots, playlists, etcetera. Refer to question 6.3.1 in the FAQ for more information about the various versions of this application. (The full installation manual for this application is available for download at the CD-i Technical Documentation - Software section on this site.)
    Download vcd_on_cdi_411.zip (392 KB)


    Photo-CD

  • Photo-CD on CD-i 3.2

    • Photo-CD on CD-i 3.2 is the latest commercialy released version of the Photo-CD runtime for CD-i. It supports all of the features of the Photo-CD specification (including playlists with sound on Portfolio Photo-CDs), and it is optimized for fast performance while displaying pictures. Refer to question 5.9.2 in the FAQ for more information about the various versions of this application.
    Download photocd_on_cdi_32.zip (275 KB)

  • Photo-CD on CD-i 3.3

    • Latest interim release of the Photo-CD on CD-i application (version 3.3), made available to Kodak for internal usage in March 1996.
    Download photocd_on_cdi_33.zip (272 KB)

  • Alternative Photo-CD application

    • This is a small alternative for the official Photo-CD application for CD-i that was and is still used on all official Kodak Photo-CDs. It does not allow for playlists and Favourite Picture Selections, but it has a zoom function that is user definable (the zoom factor can be changed).
    Download alt_photocd.zip (63 KB)
