Terms and abreviations

Documents
Most of the PDF documents are obtained from http://www.freeservicemanuals.info.
They often contain several service manuals in a single PDF document as noted in the next column.
The Sony documents are from https://www.manualscenter.com.

Service Manuals
Service manuals are shared between localized versions of the same player version as noted in this column.
See the Philips' version indication method for the version numbers of localized versions (i.e. add /05 for UK and /17 for US).

PCB
Indicates the version of the Printed Circuit Board (or mainboard) used. Some players where produced with different PCBs.
Block and circuit diagrams in the service manual are often shared between all players using the same PCB version.

Diagrams
The block and circuit diagrams in the service manuals are often shared for several players as noted in this column.

See also the Technical Training Manual, CD-I, Philips CDI 910 in the CD-i Technical Documentation section that contains a general introduction to CD-i hardware and troubleshooting and describes a lot of technical details for the Philips CDI910 player and Mini-MMC board players in general.
